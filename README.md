## 💫 About the Project

**My Little Universe** is a personal website created to introduce who I am, what I am interested in, and the ideas I hope to bring to life.

The website uses a space-inspired theme with stars, glowing objects, moving planets, glass-style cards, animations, and creative visual elements.

---

## 🪐 Website Sections

- **Main Banner** — The main introduction to the website
- **A Little About Me** — Information about my field of study and interests
- **The Worlds Inside Me** — The Builder, The Explorer, and The Dreamer
- **Ideas I Want to Bring to Life** — Ideas related to technology, health, and creative digital experiences
- **Currently Loading** — A visual representation of the skills I am developing
- **Final Message** — A closing message about continuing to learn and grow

---

## ✨ Features

- Smooth navigation between sections
- Space-themed background
- Animated stars and planets
- Floating animated words
- Glass-style cards
- Hover effects
- Animated progress bars
- Google Fonts
- Semantic HTML structure
- Organized external CSS file
